# Enclosure Files

3D-print and slicer files for the SK8Sense electronics enclosure.

```text
skatesense_base.stl
skatesense_lid.stl
electronics_base.stl
electronics_lid.stl
UM2C_electronics_enclosure.ufp
```

These files are build artifacts for the physical prototype. They are separate from the mobile app and firmware code.
